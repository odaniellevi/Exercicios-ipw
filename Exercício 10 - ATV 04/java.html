O que é o Java?

Java é uma linguagem de programação e plataforma de computação liberada pela primeira vez pela
Sun Microsystems em 1995. De um início humilde, ela evoluiu para uma grande participação no mundo digital dos dias atuais,
oferecendo a plataforma confiável na qual muitos serviços e aplicativos são desenvolvidos.
Produtos e serviços novos e inovadores projetados para o futuro continuam a confiar no Java também.
