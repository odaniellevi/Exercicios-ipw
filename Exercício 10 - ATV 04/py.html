O que é o Pyhton?

Python é uma linguagem de programação de alto nível (linguagem de computador mais próxima da humana),
criada pelo matemático holandês Guido van Rossum e publicada em 1991. Sua intenção era criar uma linguagem de programação que fosse simples,
mas ao mesmo tempo poderosa o suficiente para resolver problemas complexos. E apesar do símbolo da linguagem ser uma cobra/serpente,
seu nome não faz referência ao réptil,
mas sim ao grupo humorístico britânico Monty Python, do qual Guido continua fã até hoje. Passados quase 30 anos, a linguagem Python se tornou madura,
robusta, com uma comunidade vibrante e ativa e é cada vez mais utilizada em áreas como inteligência artificial, ciência de dados e desenvolvimento web.
Python suporta programação orientada a objeto, programação funcional
e programação imperativa.
Além disso, Python é a linguagem número 1 quando trabalhamos com machine learning.
Isso porque sua biblioteca padrão é uma das mais completas dentre todas as linguagens de programação,
incluindo módulos de acesso a banco de dados, estatística e, claro, matemática.
Desta forma, Python pode fazer mais com menos linhas de código. Atualmente, por trás da linguagem está a Python Software Foundation,
que é uma fundação sem fins lucrativos que administra os rumos da linguagem Python e sua evolução.
