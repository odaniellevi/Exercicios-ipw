O que é uma linguagem de progamação?

A linguagem de programação é um sistema de comunicação estruturado, composto por conjuntos de símbolos,
palavras-chave, regras semânticas e sintáticas que permitem o entendimento entre um programador e uma máquina.
É importante enfatizar que é um erro comum usar a linguagem de programação e a linguagem de computação
como sinônimos. Mas por que não devemos confundi-las? Bem, é porque a linguagem de programação obedece a
um conjunto de regras que permitem expressar as instruções que serão interpretadas pelo programador.
Já a linguagem de computação inclui outras linguagens que formatam um texto,
mas não podemos considerá-la uma programação em si mesma.
Portanto, nem todas as linguagens computacionais são de programação, mas todas as linguagens de programação são computacionais. Portanto, nem todas as linguagens computacionais são de programação,
mas todas as linguagens de programação são computacionais.
